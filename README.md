# README


#### サイト名：求職者、企業のマッチングサイト

#### 製品概要: サブスクリプションモデルの人材マッチングサイト
* 日系企業を中心とする外資系企業への就職
* 企業と求職者が自由にコンタクト可能

#### Version
Rails 5.2.4

Ruby 2.6.5
